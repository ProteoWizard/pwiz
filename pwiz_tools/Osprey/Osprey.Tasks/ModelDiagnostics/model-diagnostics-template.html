Null-alignment density ratio (non-parametric)

The per-class score densities above, divided: target : decoy (the diagnostic) and, when entrapment is present, p_target : p_decoy (a matched-null reference). On a fair (equal-chance) pairing the target:decoy line rides a flat plateau across the null-dominated left side, then rises where real hits begin; a sloping left side means the decoys don't track the false-target null — an equal-chance / decoy-quality violation, visible with no parametric fit (unlike the decoy-normal overlay, which misfits skewed decoys). FDR scope — experiment-wide vs per-run

Both cards below switch together between two Osprey q-value scopes. The experiment-wide q is pooled across runs (the value --fdrbench emits, so the calibration curve reproduces the FDRBench plot exactly); the per-run q gates each run on its own — the picture FDRBench collapses runs and cannot show. This is orthogonal to the page-level Pass 1 / Pass 2 switch.

FDR calibration — Osprey q vs true FDP

Entrapment-measured true false-discovery proportion versus Osprey's own q-value, computed in-process from the pairing manifest — no FDRBench install. At the experiment-wide scope this reproduces the FDRBench --fdrbench plot exactly (FDRBench just counts entrapment on the q Osprey emits); the per-run scope is a picture FDRBench can't show. Each has a [0, 2%] zoom (FDP axis auto-scaled) and a full-extent panel.

Identification yield

Accepted real-target precursors as the reported q-value threshold sweeps, at the scope selected above. The per-run curve keeps climbing (admitting non-reproducing precursors); the experiment-wide curve saturates once the real population is found — a yield that never plateaus is itself an FDR-control tell.

Calibration FDR — cal-q vs true FDP

Entrapment-measured true false-discovery proportion of the selected file's calibration anchor set versus its calibration q-value. Anchor yield

Cross-run reproducibility, gated on one q-value scope. Per-run gates each detection on its run-level q; experiment-wide also requires the experiment-wide q (max(run q, experiment q)), the value that controls dataset-wide multiple testing. Per-run FDR does not bound the experiment-wide error, so flipping to experiment-wide should shrink the k=1 bump and flatten the union (Collins et al. 2017; Rosenberger et al. 2017).

Precursor detections by run

Detections per run (bars) with the running union (unique precursors seen so far) and intersection (detected in every run so far). A fast-collapsing intersection or a runaway union is the reproducibility tell: real precursors recur, false ones don't. The dashed line marks precursors detected in at least half the runs — entrapment-free FDR QC that works on any target+decoy library.

Precursors by number of runs detected

How many precursors are detected in exactly k of the N runs. The healthy shape is J/U: a tall right peak at k=N (reproducible real precursors) with a modest k=1 bump. A growing left (k=1) bump is the FDR-struggling signature — false precursors don't reproduce, so they pile up at low k. When an entrapment library is present, the amber curve on the right axis is the entrapment-measured FDP of each k-slice — the contamination fraction of the precursors detected in exactly k runs (dashed line = the nominal run FDR). It rises at low k and falls toward the reproducible k=N peak, and it is the adjudicator for whether the singletons that survive the experiment-wide gate are real rare biology or over-admitted 1-hit-wonders.

Union FDP vs number of runs

The entrapment-measured FDP of the accumulated union of detected precursors as runs are added (1…N), shown for both q scopes at once (this panel does not follow the toggle — the gap is the point). Per-run FDR does not bound the dataset-wide error: false positives are ~random per run, so the union accretes fresh ones and its FDP climbs with run count (Collins et al. 2017; Rosenberger et al. 2017). The experiment-wide gate controls the global error, so its curve stays far flatter — the widening gap is the run-vs-global FDR problem, quantified. At the full 82-run dataset the per-run curve is expected to be materially worse. Dashed line = the nominal run FDR.

Paired decoy-win fraction

Within each target–decoy pair (shared base_id), how often the decoy outscores its target, binned by the winner's score. A fair null sits at the 50% line. Real pairs dipping below 50% at high score is native signal winning as it should; entrapment pairs holding the coin confirms the decoys are honest nulls. Marginal densities cannot see this within-pair ordering — only the competition can.
