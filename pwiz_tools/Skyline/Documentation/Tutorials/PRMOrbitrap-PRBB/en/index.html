PRM with an Orbitrap Mass Spectrometer

In this tutorial we will set up an acquisition method to quantify 31 peptides corresponding to 19 proteins of interest in murine fibroblasts using Parallel reaction monitoring (PRM). Briefly, the â€œCell cycle mouse fibroblastâ€ dataset used in this tutorial consists in murine fibroblasts in three different stages of the cell cycle, including i) G1 phase, ii) S phase, and iii) G2 plus Mitosis phases. Each condition has three biological replicates.

In this tutorial we will focus on how to set up a method for acquiring and extracting data in PRM mode.

Note: This tutorial is based in Skyline v20.2
