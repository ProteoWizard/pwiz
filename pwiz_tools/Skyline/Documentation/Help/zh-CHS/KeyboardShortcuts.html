Keyboard Shortcuts

Skyline provides two types of keyboard access to menu commands:

  • Keyboard Shortcuts Keyboard Shortcuts are single-stroke key combinations (like Ctrl+N) that can be used to perform menu actions directly. These shortcuts are displayed to the right of the menu text on the menus themselves.
  • Menu Mnemonics Mnemonics are underlined letters in menu text that become visible when you press the Alt key. You can use these letters to navigate menus using the keyboard, or learn the complete keystroke sequences to access any menu command without using the mouse.
